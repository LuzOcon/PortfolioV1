Control de Gastos

Programa para el registro de finanzas.

Sobre el proyecto

Este proyecto fue desarrollado dentro del curso InmersiónDev de Alura Latam. Está orientado a la gestión de finanzas personales. Permite al usuario registrar ingresos, egresos para obtener un balance general dinámico y realiza un guardado de estos en el LocalStorage.

Rol y Tecnologías

  • Rol: HTML, CSS, JavaScript (DOM y LocalStorage).
  • Tecnologías: HTML, CSS, JavaScript.

Desafíos y Soluciones

El principal reto fue crear una experiencia fluida donde el usuario pudiera ver el impacto de cada operación al instante, sin recargar la página. También implementé almacenamiento local para conservar los datos entre sesiones, mejorando la usabilidad del proyecto.
